# AstroAI

AstroAI is an offline-first Vedic astrology platform being built using Python, FastAPI, Swiss Ephemeris, SQLite, and Flutter.

## Current Features

### Chart Engine

* Ascendant Calculation
* 12 House Generation
* Planetary Position Calculation
* Rahu and Ketu Calculation
* Retrograde Detection
* Zodiac Sign Detection
* Nakshatra Detection
* Pada Detection
* House Placement Detection

### Dasha Engine

* Birth Mahadasha Detection
* Dasha Balance Calculation
* Current Mahadasha Calculation

### Database

* City Database Integration
* Coordinate Lookup System

## Technology Stack

### Backend
